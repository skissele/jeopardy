# Jeopardy Game

A Jeopardy-style trivia board built from a CSV dataset, with randomized categories, multiple-choice answers, and score tracking.

## Tech
- React + Vite
- CSV-driven questions

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Data
The game loads clues from `public/JEOPARDY_CSV.csv`.
